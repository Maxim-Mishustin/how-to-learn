# Как научиться
---------------

Этот сайт составлен на языке HTML и CSS. Использовалась структра БЭМ и файловая структура Nested.
В этом проекте имеется несколько блоков. Использовались анимация в 2х блоках, добавление 2х видео, 
ссылки меняющие прозрачность при наведении на них курсора. В повторяющихся блоках не дублируются
CSS-свойства.
